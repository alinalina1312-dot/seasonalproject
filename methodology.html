Как работает система прогнозирования

Система использует библиотеку Prophet для прогноза спроса по истории из Excel: колонки даты ds и значения y.

Этапы алгоритма

  1. 01 Загрузка. Excel через Colab; чтение ds, y.
  2. 02 Подготовка. Агрегация по месяцам (freq='MS') — меньше шума, стабильнее ряд: ymonth = Σi=1n yi.
  3. 03 Обучение. Prophet оценивает тренд, годовую сезонность и повторяемые колебания (multiplicative).
  4. 04 Модель. y(t) = g(t) × s(t) + εt — g(t) тренд, s(t) сезонность, εt ошибка.
  5. 05 Прогноз. yhat, нижняя и верхняя границы неопределённости.
  6. 06 Визуализация и экспорт. График и выгрузка будущего в Excel для планирования.
Google Colab · Python прокрутка 
import pandas as pd
import matplotlib.pyplot as plt
from prophet import Prophet
from google.colab import files

# 1. ЗАГРУЗКА ФАЙЛА
uploaded = files.upload()
filename = list(uploaded.keys())[0]
df = pd.read_excel(filename)

# 2. ПОДГОТОВКА ДАННЫХ
df["ds"] = pd.to_datetime(df["ds"])
# Агрегируем данные по месяцам (freq='MS').
# Это уберет хаос на графике и сделает прогноз точным.
df = df.groupby(pd.Grouper(key='ds', freq='MS'))['y'].sum().reset_index()

# 3. НАСТРОЙКА МОДЕЛИ
# seasonality_mode='multiplicative' — критично для твоего типа данных
model = Prophet(
    yearly_seasonality=True,
    weekly_seasonality=False,
    daily_seasonality=False,
    seasonality_mode='multiplicative'
)
model.fit(df)

# 4. СОЗДАНИЕ ПЕРИОДА ДЛЯ ПРОГНОЗА
# periods=24 — прогноз на 2 года вперед. Можешь менять.
future = model.make_future_dataframe(periods=24, freq="MS")
forecast = model.predict(future)

# 5. ВИЗУАЛИЗАЦИЯ
fig = model.plot(forecast)
plt.title("Прогноз спроса")
plt.axvline(x=df['ds'].max(), color='red', linestyle='--', label='Начало прогноза')  # Линия разделения
plt.legend()
plt.show()

# 6. ФОРМИРОВАНИЕ EXCEL (ТОЛЬКО БУДУЩЕЕ)
# Оставляем только то, что позже последней даты в твоем файле
history_end_date = df["ds"].max()
only_future = forecast[forecast["ds"] > history_end_date].copy()
# Выбираем нужные колонки и убираем время
result = only_future[["ds", "yhat", "yhat_lower", "yhat_upper"]].copy()
result["ds"] = result["ds"].dt.date
# Переименуем для красоты
result.columns = ['Дата', 'Прогноз', 'Мин_порог', 'Макс_порог']

# 7. СОХРАНЕНИЕ
result.to_excel("future_forecast.xlsx", index=False)
files.download("future_forecast.xlsx")
